# 정보처리기사 CBT 시험 시스템

정보처리기사 자격증 시험을 위한 Computer Based Test(CBT) 웹 애플리케이션입니다. 필기 시험과 실기 시험 두 가지 모드를 제공하며, 실제 시험 환경과 유사한 인터페이스로 연습할 수 있습니다.

## ✨ 주요 기능

### 필기 시험 모드
- **객관식 100문제**: 다양한 회차의 기출 문제 제공
- **150분 제한시간**: 실제 시험과 동일한 시간 제한
- **문제 랜덤화**: Fisher-Yates 셔플 알고리즘을 사용한 문제 순서 랜덤화
- **진행률 표시**: 현재 진행 상황을 시각적으로 확인
- **문제 네비게이션**: 문제 번호 그리드를 통한 빠른 이동
- **오답 확인**: 시험 완료 후 틀린 문제만 다시 풀기 가능
- **회차 선택**: 다양한 시험 회차 선택 가능

### 실기 시험 모드
- **키워드 문제풀이**: 주관식 답안 입력 방식
- **개념 정리 및 암기**: 실기 시험 대비용 학습 모드
- **문제 섞기**: 랜덤 순서로 문제 풀이 가능
- **리뷰 모드**: 틀린 문제 중심으로 복습

## 🛠️ 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Library**: jQuery 3.6.0
- **Data Format**: JSON
- **Styling**: Custom CSS (Gradient Design)

## 🚀 시작하기

### 요구사항

- 웹 브라우저 (Chrome, Firefox, Safari, Edge 등)
- 로컬 웹 서버 (선택사항, CORS 이슈 방지)

### 설치 및 실행

1. **프로젝트 클론 또는 다운로드**
```bash
git clone <repository-url>
cd cbt
```

2. **로컬 웹 서버 실행 (권장)**

Python 3를 사용하는 경우:
```bash
python -m http.server 8000
```

Node.js를 사용하는 경우:
```bash
npx http-server -p 8000
```

3. **브라우저에서 접속**
```
http://localhost:8000
```

또는 `index.html` 파일을 직접 브라우저에서 열 수 있습니다.

## 📖 사용 방법

### 필기 시험

1. 메인 화면에서 **"필기 시험 시작"** 버튼 클릭
2. 원하는 시험 회차 선택
3. 시험 시작 후 문제 풀이
4. 문제 네비게이션을 통해 원하는 문제로 이동 가능
5. 모든 문제를 풀거나 제출 버튼 클릭
6. 결과 확인 후 오답만 다시 풀기 가능

### 실기 시험

1. 메인 화면에서 **"실기 시험 시작"** 버튼 클릭
2. 키워드 문제 풀이 시작
3. 문제 섞기 기능으로 랜덤 순서로 학습
4. 틀린 문제는 자동으로 저장되어 복습 가능

## 🎨 주요 기능 상세

### 문제 랜덤화
- Fisher-Yates 셔플 알고리즘을 사용하여 문제와 정답을 함께 섞습니다
- 매번 시험을 시작할 때마다 다른 순서로 문제가 출제됩니다

### 타이머 기능
- 150분(9000초) 카운트다운 타이머
- 시각적으로 시간을 확인할 수 있는 UI

### 진행률 표시
- 현재 풀이한 문제 수와 전체 문제 수를 표시
- 진행률 바를 통해 시각적으로 확인 가능

### 오답 관리
- 사용자가 선택한 답안과 정답을 비교
- 틀린 문제만 필터링하여 다시 풀기 가능
- 리뷰 모드에서 틀린 문제 중심으로 학습

## 🔧 커스터마이징

### 문제 데이터 추가
1. `data/data.json` 파일에 새로운 문제 추가
2. `data/answer.json` 파일에 해당 문제의 정답 추가
3. 페이지 새로고침 후 확인

### 스타일 변경
- `css/styles.css` 파일을 수정하여 디자인 변경 가능
- 그라데이션 색상, 버튼 스타일 등 커스터마이징 가능

### 시험 시간 변경
- `js/app.js` 파일의 `timeLeft` 변수 값을 변경 (초 단위)
- 예: 120분 = 120 * 60 = 7200초

## 📝 주요 함수

- `loadExamData()`: 시험 데이터 로드 및 초기화
- `shuffleQuestions()`: 문제 순서 랜덤화
- `displayQuestion(index)`: 문제 표시
- `updateProgress()`: 진행률 업데이트
- `startTimer()`: 타이머 시작
- `submitExam()`: 시험 제출 및 결과 계산
- `showReviewMode()`: 오답 확인 모드
